# Арт Периметр — Сайт компании по установке заборов и металлоконструкций

## 📋 О Проекте

**Арт Периметр** — современный одностраничный сайт (SPA) для компании, занимающейся установкой заборов, ворот, навесов и других металлоконструкций в Москве и Московской области.

### 🔧 Технологический Стек

| Компонент | Технология |
|-----------|------------|
| **Backend** | Node.js + Express |
| **Frontend** | Vanilla JavaScript (ES6+) |
| **Стили** | CSS3 (Custom Properties, Grid, Flexbox) |
| **База данных** | JSON-файл (db.json) |
| **API** | RESTful API (Express endpoints) |

---

## 🚀 Быстрый Старт

### Требования
- Node.js v14+ 
- npm v6+

### Установка

```bash
# Клонирование репозитория
cd /workspace

# Установка зависимостей
npm install

# Запуск сервера
npm start
```

### Доступ к приложению

| Ресурс | URL |
|--------|-----|
| **Сайт** | http://localhost:3000 |
| **Админ-панель** | http://localhost:3000/admin.html |
| **Пароль администратора** | `admin123` |

## 📡 API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| `GET` | `/api/data` | Получить весь контент сайта |
| `POST` | `/api/login` | Аутентификация администратора |
| `POST` | `/api/save` | Сохранить изменения контента |
| `GET` | `/api/leads` | Получить список заявок |
| `POST` | `/api/leads` | Создать новую заявку |

### Пример запроса

```javascript
// Получение контента
fetch('/api/data')
  .then(res => res.json())
  .then(data => console.log(data));

// Отправка заявки
fetch('/api/leads', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    type: 'contact',
    name: 'Иван',
    phone: '+7 999 123 45 67'
  })
});
```
